# Predictive Analytics Using Historical Data

## 📌 Project Overview

This project demonstrates how predictive analytics can be used to forecast future sales using historical business data. 
A Linear Regression model is developed using Python and Scikit-learn to predict sales based on marketing expenditure and website visitors.

---

## 🎯 Objective

To build a machine learning model that predicts future sales from historical data and helps support business decision-making.

---

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- Matplotlib
- Scikit-learn
- OpenPyXL

---

## 📂 Dataset

The dataset contains the following columns:

- Month
- Marketing_Spend
- Website_Visitors
- Sales

---

## 📊 Project Workflow

1. Import required libraries.
2. Load the Excel dataset.
3. Explore and preprocess the data.
4. Select input and output variables.
5. Split the dataset into training and testing sets.
6. Train a Linear Regression model.
7. Predict sales on test data.
8. Evaluate the model using MAE, MSE, and R² Score.
9. Visualize Actual vs Predicted Sales.
10. Export prediction results to an Excel file.

---

## 📈 Output

- Historical data analysis
- Sales prediction using Linear Regression
- Actual vs Predicted Sales graph
- Prediction results exported to Excel
